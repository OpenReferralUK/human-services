About

Use this page to see Application Programming Interface (API) endpoints for directories outputting service data and which tools consume that data in a format compliant with the Open Referral UK standard. Tools call the API /services web method. Different endpoints support different filters, sent as parameters to the /services web method. All endpoints output JSON data describing services in the structure. Some provide extra fields of information. Some also support output in Comma Separated Values (CSV) format.
